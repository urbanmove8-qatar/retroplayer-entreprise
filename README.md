# üìº RetroPlayer Enterprise (v1.0.0)

**RetroPlayer** is a lightweight, open-source video UI wrapper designed to bring the sharp, professional "Metro" aesthetic of the IE10/Windows 8 era to modern web video. 

---

## ‚ú® Features

* **IE10 "Metro" Design:** Sharp edges, Segoe UI typography, and signature Windows Blue (`#0078d7`) theme.
* **Unified API:** Control YouTube, Vimeo, and HTML5 video with the same buttons and keyboard shortcuts.
* **Enterprise Tooling:** * **Custom Context Menu:** Right-click for system information.
    * **System Toasts:** Modern overlay notifications for volume and speed changes.
    * **Permission Tags:** Control downloads via `<downloadallow>` logic.
* **Keyboard Shortcuts:**
    * `Space` - Play / Pause
    * `M` - Mute Toggle
    * `F` - Fullscreen Toggle

## üöÄ Quick Start

1.  Include the script in your HTML header or footer.
2.  Add your video content as usual.
3.  (Optional) Use permission tags:
    ```html
    <downloadallow></downloadallow>
    ```
